About 0BTC Wire

0BTC Wire is a zero-knowledge UTXO system that enables private and scalable transactions. It uses Plonky2 for efficient proof generation and verification.

Key features:

  • Optimized hash function gadgets
  • Parallel proof generation
  • Recursive proof aggregation
  • WASM integration for browser-based applications

This demo showcases the WASM integration and recursive proof aggregation features.
